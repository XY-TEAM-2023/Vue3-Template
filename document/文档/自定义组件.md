## 搜索框

![image-20231219115902530](自定义组件.assets/image-20231219115902530.png)

```vue
<el-input-search
    v-model="searchAccount"
    placeholder="userListView.searchAccount"
    width="200"
    @search="requestUserList"
/>
```

| 字段        | 作用                      |
| ----------- | ------------------------- |
| v-model     | 搜索内容的变量            |
| placeholder | 输入框提示文本的国际化key |
| width       | 总宽度                    |
| search      | 搜索事件                  |
| max-length  | 最大长度                  |
